Introdução ao Python e Ambiente

Configure seu ambiente de desenvolvimento para Python e escreva seu primeiro "Olá, Mundo!".

O que é Python e por que é o "Canivete Suíço" dos Dados?

Python é uma linguagem de programação de alto nível, famosa por sua sintaxe limpa e legível. Pense nela como uma linguagem que se parece muito com o inglês, o que a torna relativamente fácil de aprender.

Para quem trabalha com automação e dados, Python é a ferramenta preferida por três motivos principais:

  • Simplicidade: Você pode escrever scripts poderosos com poucas linhas de código.
  • Ecossistema de Bibliotecas: Existe uma biblioteca (pacote de código pronto) para quase tudo, especialmente para dados (Pandas, NumPy), APIs (Requests) e web scraping (BeautifulSoup).
  • Versatilidade: A mesma linguagem que você usa para analisar um arquivo CSV pode ser usada para construir uma API, criar um site ou treinar um modelo de inteligência artificial.

Configurando seu Ambiente de Trabalho

Para começar a programar em Python, você precisa de duas coisas: o interpretador Python (o "cérebro" que entende a linguagem) e um bom editor de texto (a "cozinha" onde você escreve suas receitas).

Opção 1: O Caminho Rápido (Google Colab)

Para aprender e experimentar, a forma mais fácil é usar o Google Colaboratory (Colab). É um ambiente Jupyter Notebook gratuito que roda no seu navegador. Você não precisa instalar nada!

tips_and_updates
Recomendação: Para esta trilha, o Google Colab é perfeito. Ele permite que você se concentre na linguagem sem se preocupar com a instalação.

Opção 2: O Ambiente Profissional (VS Code)

Para projetos mais sérios, você vai querer um ambiente local.

  1. Instale Python: Baixe a versão mais recente em python.org. Marque a opção "Add Python to PATH" durante a instalação no Windows.
  2. Instale o VS Code: Baixe em code.visualstudio.com.
  3. Instale a Extensão Python: Dentro do VS Code, vá para a aba de Extensões e procure por "Python" da Microsoft e instale-a.

Seu Primeiro Código: "Olá, Mundo!"

A tradição ao aprender uma nova linguagem é fazer o computador dizer "Olá, Mundo!". Em Python, isso é incrivelmente simples. Usamos a função print().

hello.py

# Comentários em Python começam com '#' e são ignorados pelo interpretador.
# Eles servem para deixar anotações para nós, humanos.

# A função print() exibe na tela o que estiver dentro dos parênteses.
print("Olá, Mundo da Automação com Python!")

Teste Você Mesmo!

Agora é sua vez. Abra o Google Colab, crie um novo "notebook" e execute seu primeiro código.

  1. Acesse colab.research.google.com.
  2. Clique em "Novo notebook".
  3. Na primeira célula, digite print("Seu nome aqui!").
  4. Pressione Shift + Enter para executar.

Você verá seu nome impresso logo abaixo da célula. Parabéns, você é oficialmente um programador Python!
