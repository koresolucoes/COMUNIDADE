Tipos de Dados Essenciais e Estruturas

Entenda os blocos de construção da informação: de textos e números a listas e dicionários.

Se variáveis são "caixas" para guardar informações, os tipos de dados definem que tipo de coisa podemos guardar em cada caixa. O computador precisa saber se está lidando com um texto, um número ou uma decisão para poder realizar as operações corretas.

1. Strings: A Magia do Texto

Uma String é simplesmente um pedaço de texto. Pode ser uma letra, uma palavra, uma frase ou um livro inteiro. Pense nela como um colar de contas, onde cada conta é um caractere. Para dizer ao computador que algo é um texto, nós o envolvemos em aspas (simples ' ou duplas ").

Exemplo: Trabalhando com texto

primeiro_nome = "João"
ultimo_nome = "Silva"

# Podemos "somar" strings para juntá-las (concatenação)
nome_completo = primeiro_nome + " " + ultimo_nome

print(nome_completo)  # Mostra "João Silva" na tela

2. Numbers: O Poder dos Cálculos

Números são usados para... bem, números! Eles são a base de toda a matemática que um computador faz. Existem dois tipos principais:

  • Integers (Inteiros): Números inteiros, sem casas decimais. Ex: 10, -5, 0.
  • Floats (Ponto Flutuante): Números com casas decimais. Ex: 9.99, -0.5, 3.14.

Exemplo: Operações matemáticas

quantidade = 5       # Um Integer
preco_unitario = 9.99  # Um Float

total = quantidade * preco_unitario

print(total)  # Mostra 49.95 na tela
tips_and_updates
Importante: Você não pode fazer contas com uma string! "5" + "5" resulta na string "55", não no número 10. O computador trata texto e números de formas muito diferentes.

3. Booleanos: As Chaves da Decisão

Um Booleano é o tipo de dado mais simples de todos. Ele só pode ter dois valores: True (Verdadeiro) ou False (Falso). Pense nele como um interruptor de luz: está ligado ou desligado. Não há meio-termo.

Booleanos são o coração das condicionais (if/else), pois eles são o resultado de qualquer comparação que você faça.

Exemplo: O resultado de uma pergunta

idade = 20
e_maior_de_idade = idade >= 18  # A pergunta "20 é maior ou igual a 18?"

print(e_maior_de_idade) # Mostra "True" na tela

tem_estoque = False # Podemos definir o valor diretamente

if e_maior_de_idade:
  print("Acesso permitido.")

4. Estruturas de Dados: Caixas que Guardam Caixas

Além dos tipos básicos, podemos criar estruturas mais complexas para agrupar informações relacionadas.

Arrays (ou Listas)

Um Array (ou Lista) é uma coleção ordenada de itens. Imagine uma estante com várias gavetas numeradas. Você pode guardar um item em cada gaveta e acessá-lo pelo número da gaveta (o "índice").

Exemplo: Uma lista de compras

lista_de_compras = ["Maçãs", "Leite", "Pão"]
  
# Acessamos o primeiro item (índice 0)
print(lista_de_compras[0]) # Mostra "Maçãs"
  
# Podemos adicionar mais itens
lista_de_compras.append("Café")

Dicionários (ou Objetos)

Um Dicionário (ou Objeto) é uma coleção de pares "chave-valor". Em vez de usar um número como índice, você usa uma "chave" (uma string) para acessar um "valor". É como uma agenda de contatos: você procura pelo nome (chave) para encontrar o número de telefone (valor).

Exemplo: Descrevendo uma pessoa

pessoa = {{ '{' }}
  "nome": "Carlos",
  "idade": 42,
  "profissao": "Engenheiro"
{{ '}' }}

# Acessamos o valor usando a chave
print(pessoa["profissao"]) # Mostra "Engenheiro"

E o Char?

Em algumas linguagens mais antigas, um Char era um tipo de dado especial para um único caractere (como 'A'). Em linguagens modernas como Python e JavaScript, um `char` é simplesmente uma String com apenas um caractere. A ideia de um caractere individual ainda é importante, mas não é mais um tipo de dado separado.

Juntando Tudo: Um Exemplo Prático

Vamos usar todos os tipos de dados que aprendemos para descrever um produto de forma mais completa.

Exemplo: Descrevendo um Produto Complexo

produto = {{ '{' }}
  "nome": "Smartphone Pro X",      # String
  "preco": 3999.90,                  # Number (Float)
  "em_estoque": True,                # Boolean
  "tags": ["tecnologia", "celular"], # Array (lista de Strings)
  "especificacoes": {{ '{' }}                # Dicionário (Objeto)
    "tela": "6.7 polegadas",
    "ram": "8GB"
  {{ '}' }}
{{ '}' }}

# Agora podemos acessar informações específicas
print("O " + produto["nome"] + " custa R$ " + str(produto["preco"]))
print("RAM: " + produto["especificacoes"]["ram"])

if produto["em_estoque"]:
  print("Compre agora!")

Conclusão: A Base da Informação

Strings, Números e Booleanos são os "átomos", enquanto Arrays e Dicionários são as "moléculas" que usamos para construir informações complexas. Dominar essas estruturas é fundamental para representar qualquer dado do mundo real dentro de um programa.
