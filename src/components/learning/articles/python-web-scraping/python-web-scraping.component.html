Introdução ao Web Scraping com BeautifulSoup

Descubra como extrair informações de páginas HTML quando uma API não está disponível.

O que é Web Scraping e Quando Usá-lo?

Web Scraping (ou "raspagem de dados") é a técnica de extrair dados de websites de forma automatizada. É o seu plano B. O ideal é sempre usar uma API oficial, que fornece dados de forma estruturada e confiável. Mas e se o site que você quer extrair dados (um e-commerce, um portal de notícias) não oferece uma API?

Nesses casos, o web scraping permite que seu código "finja" ser um navegador, baixe o conteúdo HTML da página e depois o analise para encontrar e extrair as informações que você precisa.

warning
Ética e Legalidade: Sempre verifique o arquivo robots.txt de um site (ex: google.com/robots.txt) para ver as regras de automação. Não sobrecarregue os servidores com muitas requisições rápidas e nunca use scraping para atividades ilegais.

As Ferramentas: `requests` e `BeautifulSoup`

Para fazer web scraping, combinamos duas bibliotecas:

  • `requests`: Que já conhecemos. Usaremos para baixar o código-fonte HTML da página.
  • `BeautifulSoup`: Uma biblioteca fantástica para "navegar" e pesquisar dentro do HTML. Ela transforma o texto HTML bruto em um objeto Python que podemos explorar.

Instale a `BeautifulSoup` (a biblioteca se chama `beautifulsoup4`):

pip install beautifulsoup4

O Processo de Scraping em 4 Passos

Vamos usar o site books.toscrape.com, um site feito para praticar scraping, como nosso alvo.

Passo 1: Inspecionar o Alvo

Abra o site no seu navegador. Clique com o botão direito sobre o título de um livro e selecione "Inspecionar". As Ferramentas de Desenvolvedor abrirão, mostrando o código HTML. Você verá algo como: 

<h3><a href="..." title="A Light in the Attic">A Light in the ...</a></h3>

Observe que o título está dentro de uma tag <a>, que por sua vez está dentro de uma <h3>. Essa estrutura é a chave.

Passos 2, 3 e 4: Código em Ação

Exemplo: Extraindo títulos de livros

import requests
from bs4 import BeautifulSoup

url = "http://books.toscrape.com/"

# PASSO 2: Baixar o HTML
response = requests.get(url)
html_content = response.text

# PASSO 3: "Alimentar" o BeautifulSoup com o HTML
soup = BeautifulSoup(html_content, 'html.parser')

# PASSO 4: Encontrar os elementos e extrair os dados
# Usamos 'find_all' para encontrar todas as tags <h3>
# Em seguida, iteramos sobre cada <h3> encontrado
titulos = []
for h3_tag in soup.find_all('h3'):
    # Dentro de cada <h3>, encontramos a tag <a> e pegamos seu atributo 'title'
    titulo_livro = h3_tag.find('a')['title']
    titulos.append(titulo_livro)

# Imprime os 5 primeiros títulos encontrados
for titulo in titulos[:5]:
    print(titulo)

Teste Você Mesmo!

Agora, um desafio. O site quotes.toscrape.com é outro playground para scraping.

  1. Use o que aprendeu para fazer o scraping de todas as citações da primeira página.
  2. Dica: Inspecione o elemento. Você verá que cada citação está dentro de uma tag <span> com a classe (class_) "text".
  3. Tente usar soup.find_all('span', class_='text') e, para cada item encontrado, extraia o texto com .text.
