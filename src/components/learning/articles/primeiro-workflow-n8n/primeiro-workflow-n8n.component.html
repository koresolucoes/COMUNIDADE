Prática: Seu Primeiro Workflow (Simples)

Vamos construir! Crie seu primeiro workflow "Olá, Mundo" para aprender a adicionar, conectar e executar nós na prática.

A melhor maneira de aprender é fazendo. Nesta prática, vamos criar o workflow mais simples possível para nos familiarizarmos com a mecânica de adicionar, conectar e executar nós no n8n.

O Objetivo

Vamos criar um workflow que, quando executado manualmente, gera uma pequena estrutura de dados (um JSON) com a mensagem "Olá, Mundo da Automação!".

Para isso, usaremos apenas dois nós:

  • Start Node (Gatilho): O ponto de partida padrão para qualquer novo workflow.
  • Set Node (Ação): Um nó extremamente útil para criar ou modificar dados manualmente.

Passo 1: Começando com o Nó "Start"

Ao criar um novo workflow no n8n, ele já vem com um Start Node no Canvas. Este é o nosso gatilho manual. Ele simplesmente diz: "Comece o workflow quando eu clicar no botão de execução". Não precisamos configurar nada nele.

Passo 2: Adicionando o Nó "Set"

  1. Clique no ícone de "+" no painel de nós à esquerda.
  2. Na barra de busca que aparece, digite Set.
  3. Clique e arraste o nó "Set" para o Canvas, posicionando-o à direita do nó "Start".

Passo 3: Conectando os Nós

  1. Passe o mouse sobre o círculo de saída (à direita) do nó "Start".
  2. Clique e arraste a "linha" que aparece até o círculo de entrada (à esquerda) do nó "Set".
  3. Solte o mouse. Os dois nós agora estão conectados, indicando a ordem de execução.

Passo 4: Configurando o Nó "Set"

  1. Clique no nó "Set" para abrir seu painel de parâmetros à direita.
  2. Por padrão, o modo ("Mode") está como "Append". Isso significa que ele vai adicionar os dados que criarmos aos dados que vêm do nó anterior.
  3. O modo "Keep Only Set" faria com que apenas os dados criados aqui fossem passados adiante, descartando o que veio antes. Para este exemplo, "Append" está ótimo.

Passo 5: Executando e Vendo o Resultado

  1. Clique no botão "Execute Workflow" no canto inferior esquerdo da tela.
  2. O n8n executará o fluxo. Você verá um pequeno ícone de sucesso verde sobre cada nó.
  3. Clique no nó "Set". Agora, no painel de resultados (que substitui o painel de parâmetros), você verá as abas "Input" e "Output".
  4. Clique na aba "Output". Você verá o JSON que acabamos de criar!
O "Input" mostra os dados que o nó recebeu do nó anterior. O "Output" mostra os dados que ele está enviando para o próximo nó.

Parabéns!

Você acabou de construir e executar seu primeiro workflow! Você aprendeu o ciclo fundamental de desenvolvimento no n8n: adicionar, conectar, configurar e testar. Com essa base, você está pronto para explorar gatilhos mais complexos, como Webhooks e CRON, e nós de ação que se conectam a serviços reais.
