Além dos Retângulos: Formas com Clip-Path

Descubra como a propriedade `clip-path` pode ser usada para criar layouts criativos, cortando elementos em formas geométricas.

A Tesoura do CSS

A web é feita de caixas (o Box Model). Tudo é retangular. A propriedade clip-path nos permite quebrar essa monotonia. Ela define uma região de recorte: tudo o que está dentro da forma é mostrado, tudo o que está fora é escondido (mas continua ocupando espaço no layout).

1. Formas Básicas: Círculos e Recortes

As funções mais simples são circle(), ellipse() e inset(). Elas são responsivas e fáceis de animar.

2. Polígonos: O Controle Total

A função polygon() permite definir múltiplos pares de coordenadas X Y (separados por vírgula) para desenhar qualquer forma com linhas retas.

Escolha uma Forma:

clip-path: {{ polyClip() }};

3. Animando o Recorte

Como clip-path é animável, podemos criar efeitos de revelação incríveis. O segredo é garantir que a forma inicial e final tenham o mesmo número de pontos (especialmente para polígonos).
