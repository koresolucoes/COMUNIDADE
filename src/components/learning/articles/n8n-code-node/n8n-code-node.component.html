Lógica Avançada: O Poder do Nó "Code"

Vá além das expressões simples. Aprenda a escrever código JavaScript para realizar transformações de dados complexas que outros nós não conseguem.

Quando os Nós Não São Suficientes

O n8n é fantástico com sua abordagem "low-code". Para 90% das tarefas, os nós Set, IF e Merge são suficientes para manipular seus dados. Mas e se você precisar de uma lógica mais complexa?

  • Calcular uma pontuação com base em múltiplas variáveis.
  • Fazer um loop dentro de um loop.
  • Formatar os dados de uma maneira muito específica que uma API exige.

Para esses casos, em vez de usar 10 nós "Set" e "IF" em sequência, você pode usar um único e poderoso Nó Code.

O que é o Nó Code?

O nó Code é uma "caixa de areia" onde você pode escrever código JavaScript. Ele te dá acesso total aos dados de entrada e permite que você retorne novos dados, modificados da forma que precisar.

Dentro do nó, você tem acesso aos mesmos dados que teria em uma expressão, mas com a flexibilidade de usar loops, funções, variáveis e toda a potência do JavaScript.

Acessando Dados e Retornando Resultados

O nó Code funciona com base em um princípio simples:

Estrutura Básica

// 'items' é um array com todos os itens que chegam ao nó.
// Geralmente, trabalhamos com um item de cada vez.
const item = items[0];

// Acesse os dados do item, como faria em uma expressão.
const nome = item.json.nome;
const valor = item.json.valor;

// *** Faça sua lógica complexa aqui ***
const nomeMaiusculo = nome.toUpperCase();
const valorComTaxa = valor * 1.1;

// Modifique o item original ou crie um novo.
item.json.nome_maiusculo = nomeMaiusculo;
item.json.valor_final = valorComTaxa;

// RETORNE o array de itens modificados. Isso é crucial!
return items;

O que você return no final do código é o que será passado para o próximo nó na esteira de produção.

Conclusão: O Melhor dos Dois Mundos

O nó Code é a ponte perfeita entre o "low-code" e o "pro-code". Ele permite que você construa a maior parte do seu workflow visualmente e, para aquele 10% de lógica super específica, você pode "mergulhar" no código para ter controle total. Dominar o nó Code é o que separa um usuário básico de um automatizador avançado no n8n.
