Tomando Decisões com o Nó "IF"

Aprenda a criar ramificações e lógicas condicionais em seus workflows, executando diferentes ações com base nos dados.

A Encruzilhada da Automação

Nem todo workflow é uma linha reta. Muitas vezes, você precisa que sua automação tome decisões: "Se o valor da compra for maior que R$100, envie um cupom de frete grátis. Senão, apenas envie o e-mail de confirmação."

O nó IF é a ferramenta que cria essas "encruzilhadas" no seu workflow. Ele avalia uma ou mais condições e, com base no resultado, direciona os dados para diferentes caminhos.

Como o Nó IF Funciona

O nó IF tem uma entrada (à esquerda) e duas saídas (à direita): uma para true (verdadeiro) e outra para false (falso).

No painel de parâmetros, você define uma condição. Para cada item que chega ao nó, ele testa a condição:

  • Se a condição for verdadeira, o item sai pela saída true.
  • Se a condição for falsa, o item sai pela saída false.

Construindo uma Condição

Uma condição no nó IF tem três partes principais:

Exemplo: Verificando o Valor de um Pedido

Imagine que o nó anterior te deu um JSON com {{ '{' }} "valor": 150 {{ '}' }}.

  • Value 1: Aqui você coloca uma expressão para pegar o dado que quer testar.
    {{ '{{' }} $json.valor {{ '}}' }}
  • Operation: O tipo de comparação que você quer fazer.
    Greater Than (Maior que)
  • Value 2: O valor com o qual você quer comparar.
    100

Neste caso, como 150 é maior que 100, a condição é verdadeira e o item será enviado para a saída true.

Conclusão: Dando Inteligência ao seu Workflow

O nó IF é o que transforma um simples roteiro em uma automação inteligente. Ele permite que seus workflows se adaptem a diferentes cenários e executem as ações corretas para cada situação, tornando suas automações muito mais poderosas e flexíveis.
