Variáveis e Tipos de Dados Primitivos

Domine os tipos de dados básicos: strings para texto, int e float para números, e booleanos para lógica.

O que são Variáveis? Caixas com Etiquetas

Um programa precisa lidar com informações. Uma variável é como uma caixa com uma etiqueta, onde você pode guardar um dado. A "etiqueta" é o nome da variável, e o conteúdo é o valor que ela armazena. Em Python, criar uma variável é tão simples quanto dar um nome e atribuir um valor com o sinal de igual (=).

1. Strings (`str`): O Mundo do Texto

Uma String (ou `str`) é usada para armazenar texto. Você pode criá-la usando aspas simples (') ou duplas (").

Exemplo: Manipulando texto

primeiro_nome = "Maria"
profissao = 'Engenheira de Dados'

# Podemos "somar" strings para juntá-las (concatenação)
apresentacao = "Nome: " + primeiro_nome + ", Profissão: " + profissao
print(apresentacao)

# Strings têm "métodos" úteis, como deixar tudo em maiúsculo
print(profissao.upper())  # Saída: ENGENHEIRA DE DADOS

f-strings: A Forma Moderna de Formatar Texto

Juntar strings com `+` pode ser confuso. As f-strings (strings formatadas) são a maneira mais limpa e legível de incluir variáveis dentro de um texto.

# Comece a string com a letra 'f' e coloque as variáveis entre chaves {{ '{' }}{{ '}' }}
resumo = f"{{'{'}}primeiro_nome{{'}'}} é uma {{'{'}}profissao{{'}'}}."
print(resumo)  # Saída: Maria é uma Engenheira de Dados.

2. Números (`int` e `float`): A Base dos Cálculos

Python tem dois tipos numéricos principais:

  • Inteiros (`int`): Números inteiros, sem casas decimais (ex: 10, -50).
  • Ponto Flutuante (`float`): Números com casas decimais (ex: 19.99, 3.14).

Exemplo: Operações matemáticas

ano_nascimento = 1990
altura = 1.75

idade = 2024 - ano_nascimento
imc = 70 / (altura * altura)  # O resultado da divisão é sempre um float

print(f"Idade: {{'{'}}idade{{'}'}} anos")
print(f"IMC: {{'{'}}imc{{'}'}}")

3. Booleanos (`bool`): O Coração da Lógica

Um Booleano (`bool`) representa um de dois valores: True (Verdadeiro) ou False (Falso). Eles são o resultado de comparações e a base para a tomada de decisões em condicionais.

Exemplo: Comparações

idade = 25
tem_ingresso = True

pode_entrar = idade >= 18 and tem_ingresso

print(f"Pode entrar na festa? {{'{'}}pode_entrar{{'}'}}")  # Saída: Pode entrar na festa? True

# Operadores de comparação comuns: == (igual), != (diferente), >, <, >=, <=
saldo = 100
compra_aprovada = saldo > 50
print(f"Compra aprovada? {{'{'}}compra_aprovada{{'}'}}") # Saída: Compra aprovada? True
