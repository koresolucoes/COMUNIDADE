Variáveis e Tipos de Dados

Domine a base da programação: como o computador armazena textos, números e decisões na memória.

O Conceito da "Caixa com Etiqueta"

Imagine a memória do computador como um enorme armazém. Para não perder as informações, guardamos os dados dentro de "caixas" e colamos uma etiqueta nelas.

  • A Variável é a etiqueta (o nome que você dá).
  • O Valor é o que está dentro da caixa.
  • O Tipo de Dado define o formato da caixa (se cabe texto, número, etc).

1. O Inspetor de Tipos

Em Python, você não precisa dizer o tipo da variável (como em Java ou C++). O Python descobre sozinho. Isso se chama Tipagem Dinâmica. Teste abaixo:

=

Dica: Tente "Texto", 10, 10.5 ou True.

{{ visualRepresentation() }}
Interpretação do Python:
type({{ variableName() }}) == <class '{{ detectedType() }}'>
abc

String (str)

Texto. Sempre envolto em aspas simples ' ' ou duplas " ".

nome = "Kore"
123

Numbers

int: Inteiros (10, -5).
float: Decimais (10.5, 3.14). Use ponto, não vírgula!

preco = 19.99
toggle_on

Boolean (bool)

Lógica pura. Apenas dois valores possíveis: True ou False.

ativo = True

2. O Poder das f-strings

Antigamente, juntar texto e variáveis (concatenação) era chato usando o sinal de +. Hoje, usamos f-strings. Basta colocar um f antes das aspas e usar chaves {{ '{' }} {{ '}' }}.

# Código Python:
{{ fStringCode() }}
# Resultado na tela:
{{ fStringResult() }}

3. O Perigo dos Tipos (Casting)

Para o Python, 10 (número) é diferente de "10" (texto). Somar números faz matemática. Somar textos junta as palavras. Tente misturar os tipos abaixo e veja o que acontece.

+
@if(castingResult().type === 'error') {

>>> {{ castingResult().code }}

error {{ castingResult().desc }}

} @else {

>>> {{ castingResult().code }}

{{ castingResult().val }}

{{ castingResult().desc }}
}

Como corrigir? Use as funções de casting:

  • int("10") -> 10
  • float("10.5") -> 10.5
  • str(10) -> "10"

Resumo

  • Variáveis guardam dados. O Python descobre o tipo automaticamente.
  • Use Strings para texto, Int/Float para números, Bool para lógica.
  • Use f-strings para criar textos dinâmicos.
  • Cuidado ao misturar tipos! Converta-os (casting) se necessário.
