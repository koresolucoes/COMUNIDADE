Construindo sua Primeira API REST (CRUD)

Crie os endpoints básicos para Criar, Ler, Atualizar e Deletar (CRUD) dados em seu banco através da sua API.

Agora que nosso servidor Node.js sabe como se conectar ao banco de dados, vamos ensiná-lo a fazer as quatro operações fundamentais que toda aplicação precisa: CRUD (Create, Read, Update, Delete). Vamos construir uma API para gerenciar uma lista de tarefas.

2. O Ingrediente Secreto: `express.json()`

Para que nossa API consiga ler os dados enviados nos corpos das requisições `POST` e `PUT` (que virão em formato JSON), precisamos adicionar uma linha de "configuração" ao nosso index.js. Essa linha é chamada de middleware.

Adicione app.use(express.json()); logo após a linha const app = express();.

index.js (início)

3. Construindo as Rotas (Endpoints)

Agora, vamos adicionar cada uma das operações CRUD como um endpoint no nosso index.js.

Read (Ler): GET /tarefas e GET /tarefas/:id

Vamos criar duas rotas GET: uma para listar todas as tarefas e outra para buscar uma tarefa específica pelo seu ID.

Create (Criar): POST /tarefas

Esta rota receberá um JSON no corpo da requisição com a descrição da nova tarefa e a inserirá no banco.

Update (Atualizar): PUT /tarefas/:id

Esta rota atualizará uma tarefa existente, identificada pelo ID na URL.

Delete (Deletar): DELETE /tarefas/:id

Finalmente, esta rota removerá uma tarefa do banco de dados.
