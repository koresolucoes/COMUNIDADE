Introdução ao Pandas: Series e DataFrames

Conheça as duas estruturas de dados fundamentais do Pandas e como elas representam dados tabulares.

O que é Pandas? O "Excel" do Python

Pandas é a biblioteca mais importante para ciência e análise de dados em Python. Pense nela como um "Excel com superpoderes" que você controla através de código. Ela permite carregar, manipular, limpar, analisar e visualizar dados tabulares de forma eficiente.

O nome "Pandas" vem de "Panel Data" (Dados em Painel), um termo de econometria para conjuntos de dados multidimensionais.

Instalação e Importação

Se você ainda não tem o Pandas, instale-o com o pip. A convenção da comunidade é importar a biblioteca com o apelido pd.

pip install pandas
import pandas as pd

1. A Coluna: `Series`

A primeira estrutura de dados fundamental do Pandas é a `Series`. Ela representa uma única coluna de dados, como uma coluna em uma planilha. É um array unidimensional com rótulos (chamados de índice).

Exemplo: Criando uma `Series` de nomes

import pandas as pd

nomes = ["Ana", "Bruno", "Carla"]
serie_nomes = pd.Series(nomes)

print(serie_nomes)

Saída:

0      Ana
1    Bruno
2    Carla
dtype: object

Note que o Pandas adicionou automaticamente um índice numérico (0, 1, 2) à esquerda.

2. A Tabela: `DataFrame`

A estrutura de dados mais importante e mais usada é o `DataFrame`. Ele representa uma tabela de dados bidimensional, com linhas e colunas, exatamente como uma planilha. Cada coluna de um `DataFrame` é uma `Series`.

A forma mais comum de criar um `DataFrame` do zero é a partir de um dicionário Python, onde as chaves são os nomes das colunas e os valores são as listas de dados.

Exemplo: Criando um `DataFrame` de usuários

import pandas as pd

dados = {{ '{' }}
    "nome": ["Ana", "Bruno", "Carla"],
    "idade": [28, 35, 22],
    "cidade": ["São Paulo", "Rio de Janeiro", "Belo Horizonte"]
{{ '}' }}

df = pd.DataFrame(dados)

print(df)

Saída:

    nome  idade          cidade
0    Ana     28       São Paulo
1  Bruno     35  Rio de Janeiro
2  Carla     22  Belo Horizonte

Inspecionando seu DataFrame

Depois de carregar ou criar um `DataFrame`, as primeiras coisas que você vai querer fazer são inspecioná-lo para entender seus dados.

Comandos essenciais de inspeção

# Mostra as 5 primeiras linhas do DataFrame
print(df.head())

# Mostra informações sobre o DataFrame, incluindo tipos de dados e valores nulos
print(df.info())

# Gera estatísticas descritivas para as colunas numéricas (contagem, média, desvio padrão, etc.)
print(df.describe())
