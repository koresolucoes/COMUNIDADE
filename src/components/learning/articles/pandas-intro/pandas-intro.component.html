1. A Coluna: `Series`

A primeira estrutura de dados fundamental do Pandas é a `Series`. Ela representa uma única coluna de dados, como uma coluna em uma planilha.

2. A Tabela: `DataFrame`

A estrutura de dados mais importante e mais usada é o `DataFrame`. Ele representa uma tabela de dados bidimensional, com linhas e colunas, exatamente como uma planilha.
