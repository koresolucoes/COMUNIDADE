Lógica de Programação: Variáveis, Condicionais e Loops

Aprenda os três pilares da lógica: como armazenar informações (variáveis), tomar decisões (condicionais) e repetir tarefas (loops).

Se a programação é como escrever uma receita, a lógica de programação é a gramática que nos permite criar instruções claras e sem ambiguidades. Vamos explorar os três blocos de construção fundamentais de qualquer programa.

1. Variáveis: Caixas para Guardar Coisas

Um programa precisa lidar com informações: o nome de um usuário, a idade de alguém, o preço de um produto. Uma variável é como uma caixa com uma etiqueta. Você dá um nome à caixa (a etiqueta) e guarda algo dentro dela.

Exemplo: Guardando informações

# Criamos uma "caixa" chamada 'nome' e guardamos o texto "Ana" dentro dela.
nome = "Ana"

# Criamos outra "caixa" chamada 'idade' e guardamos o número 30.
idade = 30

# Agora podemos usar as etiquetas para ver o que há dentro.
print(nome)   # Mostra "Ana" na tela
print(idade)  # Mostra 30 na tela
tips_and_updates
Dica: Dê nomes claros e descritivos às suas variáveis. Use preco_produto em vez de p. Isso torna seu código muito mais fácil de entender no futuro!

2. Condicionais: Encruzilhadas no Caminho

Programas não são apenas uma lista reta de tarefas; eles precisam tomar decisões. Condicionais (geralmente usando if, else - "se", "senão") são como encruzilhadas na sua receita. "Se uma condição for verdadeira, siga por este caminho. Senão, siga por aquele outro."

Exemplo: Verificando a idade para entrar em uma festa

idade = 20

# A condição é "idade >= 18" (idade é maior ou igual a 18?)
if idade >= 18:
  print("Pode entrar na festa!")  # Este caminho é seguido se a condição for verdadeira.
else:
  print("Desculpe, entrada permitida apenas para maiores de 18.") # Este, se for falsa.

3. Loops: Repetindo Tarefas Automaticamente

Imagine que você precisa escrever "Olá!" na tela 5 vezes. Você poderia escrever print("Olá!") 5 vezes, mas e se fossem 1000 vezes? Loops (ou laços de repetição) resolvem isso. Eles executam um bloco de código repetidamente, até que uma condição de parada seja atingida.

Exemplo: Contando de 1 a 5

Este código diz: "Para cada número no intervalo de 1 a 5, mostre esse número na tela."

for numero in range(1, 6):  # O 6 não é incluído, então vai de 1 a 5
  print(numero)

# Saída na tela:
# 1
# 2
# 3
# 4
# 5
warning
Cuidado: Loops infinitos! Se a condição de parada de um loop nunca for atingida, o programa ficará preso, executando o mesmo código para sempre e travando.

Juntando Tudo: Um Exemplo Prático

Vamos combinar os três conceitos para resolver um problema simples: verificar uma lista de convidados e imprimir quem pode entrar na festa.

Exemplo: Filtro de Festa

# 1. VARIÁVEIS: Armazenamos a lista de convidados em uma variável
convidados = [
  {{ '{' }} "nome": "Ana", "idade": 25 {{ '}' }},
  {{ '{' }} "nome": "Bruno", "idade": 17 {{ '}' }},
  {{ '{' }} "nome": "Carla", "idade": 30 {{ '}' }}
]

# 2. LOOP: Passamos por cada convidado na lista, um de cada vez
for convidado in convidados:

  # 3. CONDICIONAL: Verificamos a idade de cada convidado
  if convidado["idade"] >= 18:
    print(convidado["nome"] + " pode entrar.")
  else:
    print(convidado["nome"] + " não pode entrar.")

# Saída na tela:
# Ana pode entrar.
# Bruno não pode entrar.
# Carla pode entrar.

Conclusão: Os Blocos de Construção

Parabéns! Você acabou de aprender os três pilares que sustentam toda a programação. Não importa quão complexo um software pareça, no fundo, ele é apenas uma combinação sofisticada de variáveis para guardar dados, condicionais para tomar decisões e loops para executar repetições.
