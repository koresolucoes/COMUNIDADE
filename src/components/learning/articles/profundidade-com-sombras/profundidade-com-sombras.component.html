Profundidade e Efeitos de Vidro

Domine o uso de sombras (box-shadow) e desfoque de fundo (glassmorphism) para criar interfaces modernas e hierárquicas.

O Mundo Real Não é Plano

Em interfaces digitais, a "profundidade" (o eixo Z) é simulada. Usamos pistas visuais para dizer ao cérebro do usuário: "Este botão está acima do fundo, então ele é clicável" ou "Este modal está flutuando sobre o conteúdo". As duas principais ferramentas para isso são as Sombras e, mais recentemente, o efeito de Vidro (Glassmorphism).

1. A Anatomia da `box-shadow`

Muitos desenvolvedores copiam sombras prontas porque a sintaxe do CSS parece mágica. Vamos desmistificar isso. Ajuste os sliders para ver o que cada número realmente faz.

Quanto maior o blur, mais "alto" o elemento parece estar.

O segredo de sombras boas: opacidade baixa (0.1 a 0.3).

Objeto
box-shadow: {{ shadowX() }}px {{ shadowY() }}px {{ shadowBlur() }}px {{ shadowSpread() }}px rgba(0,0,0,{{ shadowOpacity() }});

2. A Regra das Camadas (Layering)

No mundo real, sombras não são blocos sólidos pretos. Elas são difusas. Para criar sombras realistas em CSS, raramente usamos apenas uma definição. Usamos múltiplas camadas de sombra: uma pequena e nítida para definir a borda, e uma grande e difusa para dar a sensação de altura.

3. O Efeito Vidro (Glassmorphism)

Uma tendência moderna para criar profundidade sem escurecer a interface com sombras pesadas é o Glassmorphism. A ideia é fazer com que elementos da UI pareçam vidro fosco (frosted glass), flutuando sobre o fundo. Isso é feito combinando transparência no fundo e o filtro backdrop-filter: blur().
