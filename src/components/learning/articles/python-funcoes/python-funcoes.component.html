Funções e Modularização em Python

Escreva código limpo e reutilizável criando suas próprias funções e importando módulos.

O Princípio DRY: "Don't Repeat Yourself"

Imagine que você precisa calcular o imposto sobre um valor em vários lugares do seu código. Você poderia escrever a fórmula valor * 0.07 toda vez. Mas e se o imposto mudar para 8%? Você teria que encontrar e corrigir a fórmula em todos os lugares, um processo chato e propenso a erros.

As funções resolvem isso. Uma função é um bloco de código nomeado e reutilizável que realiza uma tarefa específica. É como criar sua própria mini-ferramenta.

1. Anatomia de uma Função

Uma função em Python tem três partes principais:

  • Definição (`def`): A palavra-chave def, seguida pelo nome da função e parênteses.
  • Parâmetros (Entradas): Variáveis dentro dos parênteses que a função recebe como entrada.
  • Retorno (`return`): O resultado que a função devolve após fazer seu trabalho.

Exemplo: Função para calcular imposto

# Definindo a função
def calcular_imposto(valor_base):
    """
    Esta é uma docstring. Ela explica o que a função faz.
    Calcula um imposto de 7% sobre um valor base.
    """
    taxa_imposto = 0.07
    imposto = valor_base * taxa_imposto
    return imposto

# Usando (ou "chamando") a função
preco_produto_a = 100
imposto_a = calcular_imposto(preco_produto_a)
print(f"O imposto para o produto A é: R$ {{'{'}}imposto_a{{'}'}}") # Saída: O imposto para o produto A é: R$ 7.0

preco_produto_b = 50
imposto_b = calcular_imposto(preco_produto_b)
print(f"O imposto para o produto B é: R$ {{'{'}}imposto_b{{'}'}}") # Saída: O imposto para o produto B é: R$ 3.5
Agora, se o imposto mudar, você só precisa editar a variável taxa_imposto em um lugar: dentro da função!

2. Modularização: Importando Ferramentas

Você não precisa escrever todas as funções do zero. Python vem com uma vasta "biblioteca padrão" cheia de ferramentas prontas, organizadas em módulos. Para usar as ferramentas de um módulo, você precisa importá-lo com a palavra-chave import.

Exemplo: Usando o módulo `math` e `random`

import math
import random

# Usando uma função do módulo math para arredondar um número para cima
numero = 9.2
arredondado_para_cima = math.ceil(numero)
print(f"O número {{'{'}}numero{{'}'}} arredondado para cima é: {{'{'}}arredondado_para_cima{{'}'}}") # Saída: 10

# Usando uma função do módulo random para escolher um item aleatório de uma lista
opcoes = ["pedra", "papel", "tesoura"]
escolha_do_computador = random.choice(opcoes)
print(f"O computador escolheu: {{'{'}}escolha_do_computador{{'}'}}")
