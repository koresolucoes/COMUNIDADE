Funções e Modularização

Escreva código limpo e reutilizável criando suas próprias mini-ferramentas.

O Princípio DRY: "Don't Repeat Yourself"

As funções são blocos de código nomeados e reutilizáveis. Elas recebem entradas (parâmetros), processam algo e devolvem uma saída (retorno).

functions Laboratório de Função

# Definição da Função

def calcular_total(valor, taxa):
    imposto = valor * (taxa / 100)
    total = valor + imposto
    return total

# Chamada da Função
