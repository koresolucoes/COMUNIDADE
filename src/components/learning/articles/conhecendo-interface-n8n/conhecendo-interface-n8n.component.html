Conhecendo a Interface (O Canvas)

Faça um tour pela interface do n8n: aprenda sobre o Canvas, o painel de nós e os botões essenciais de execução e salvamento.

Ao abrir o n8n para criar um novo workflow, você é apresentado à sua principal área de trabalho. Vamos desvendar seus componentes para que você se sinta em casa.

O Mapa da Mina: A Interface Principal

A imagem abaixo destaca as 5 áreas mais importantes da interface do editor de workflows.

  1. O Canvas: Esta é a sua tela em branco, sua área de trabalho principal. É aqui que você vai arrastar e conectar os nós para construir o seu fluxograma de automação.
  2. Painel de Nós: A sua caixa de ferramentas. Aqui você pode pesquisar por qualquer nó (ex: "Google Sheets", "HTTP Request") e arrastá-lo para o Canvas.
  3. Controles do Workflow: Aqui ficam os botões vitais para gerenciar seu workflow.
    • Nome do Workflow: Clique para renomear seu fluxo para algo descritivo.
    • Save: Salva o progresso do seu trabalho.
    • Interruptor Ativo/Inativo: Liga ou desliga seu workflow. Um workflow inativo não será executado, mesmo que seu gatilho seja acionado.
  4. Controles de Execução: Estes botões são usados para testar seu workflow enquanto você o constrói.
    • Execute Workflow: Inicia uma execução de teste de todo o workflow, começando pelo nó de gatilho.
  5. Painel de Configurações (Nó/Workflow): Quando você seleciona um nó no Canvas, este painel aparece à direita, mostrando todos os parâmetros e configurações daquele nó específico. Se nenhum nó estiver selecionado, ele mostra as configurações gerais do workflow.

Conclusão: Sua Nova Oficina

A interface do n8n foi projetada para ser intuitiva. O fluxo de trabalho é simples: encontre um nó no painel esquerdo, arraste-o para o Canvas, clique nele para abrir suas configurações no painel direito, conecte-o a outros nós e use os botões de execução para testar. Com esta visão geral, você está pronto para entender a peça mais importante de todas: o nó.
