Efeitos Visuais com a Propriedade Filter

Aprenda a aplicar efeitos como desfoque, sépia e escala de cinza para manipular a aparência de imagens e elementos.

O "Photoshop" do CSS

A propriedade filter do CSS é como ter um estúdio de edição de imagem no navegador. Ela permite aplicar efeitos gráficos a elementos (normalmente imagens, mas funciona em qualquer coisa) antes que eles sejam desenhados na tela. Isso é incrivelmente útil para criar estados de hover, ajustar fotos enviadas por usuários ou criar efeitos artísticos como o popular "Glassmorphism" (usando o primo backdrop-filter).

1. Laboratório de Filtros

2. Casos de Uso Comuns

Filtros não servem apenas para fazer fotos bonitas. Eles têm utilidade prática em UI Design.

Desativado (Grayscale)

Use grayscale(100%) e opacity(50%) para indicar que um elemento (como um avatar de usuário offline) está inativo sem precisar trocar a imagem.

Modo Escuro (Invert)

Em casos extremos, invert(100%) pode transformar ícones pretos em brancos para um tema escuro instantâneo.
