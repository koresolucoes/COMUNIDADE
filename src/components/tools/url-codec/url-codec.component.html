A codificação de URL é crucial ao construir URLs dinâmicas no nó HTTP Request, especialmente para parâmetros de consulta (query parameters).

Caso de Uso: Passar JSON como parâmetro Imagine que você precisa passar um objeto JSON complexo como um parâmetro de URL. Você não pode simplesmente colocá-lo na URL. Primeiro, você precisa convertê-lo para string e depois codificá-lo.

O n8n se encarrega de chamar a função encodeURIComponent do JavaScript para garantir que todos os caracteres especiais (espaços, &, ", {, etc.) sejam transmitidos corretamente.

A biblioteca padrão urllib.parse contém as funções quote e unquote para isso.

As funções globais encodeURIComponent() e decodeURIComponent() são a forma nativa de lidar com a codificação de URLs.
