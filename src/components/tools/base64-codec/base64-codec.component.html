A manipulação de arquivos binários e Base64 é fundamental no n8n, especialmente ao lidar com APIs que enviam ou recebem arquivos.

Arquivo para Base64

Se um nó anterior (como "Read Binary File") fornece um dado binário, você pode convertê-lo para Base64 no nó "Code" para enviar em um corpo JSON:

items[0].binary.data.toString('base64');

Base64 para Arquivo

Se você recebe uma string Base64 (ex: de uma API), pode convertê-la em um arquivo binário para usar em outros nós (como "Write Binary File" ou "Send Email"):

const base64String = items[0].json.base64Data;
const fileBuffer = Buffer.from(base64String, 'base64');
const binaryData = await $binary.create(fileBuffer, 'meu_arquivo.pdf', 'application/pdf');
return {{ '{' }} json: {{ '{' }}{{ '}' }}, binary: {{ '{' }} data: binaryData {{ '}' }} {{ '}' }};
