O n8n opera nativamente com JSON, e o nó Code é seu melhor amigo para manipulações complexas.

Principais Operações:

  • Analisar (Parse): Se você recebe uma string JSON de uma API (ex: no corpo de um webhook), pode convertê-la em um objeto n8n com a expressão: {{ '{{' }} JSON.parse($json.body) {{ '}}' }}.
  • Formatar (Stringify): Para enviar um objeto n8n como uma string JSON formatada para outra API, use: {{ '{{' }} JSON.stringify($json, null, 2) {{ '}}' }}. O '2' indica a indentação, criando uma saída legível.
  • Minificar (Stringify): Para uma versão compacta, sem espaços, ideal para economizar banda, omita os argumentos extras: {{ '{{' }} JSON.stringify($json) {{ '}}' }}.
Atenção: Sempre envolva operações de `JSON.parse` em um bloco `try...catch` dentro do nó "Code" para tratar casos em que a string de entrada não é um JSON válido e evitar que seu workflow falhe.

O módulo json é nativo do Python e oferece todas as funções necessárias.

O objeto global JSON é a forma padrão de interagir com JSON em JavaScript e Node.js.
