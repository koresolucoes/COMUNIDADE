Importante sobre Segurança Este método APENAS DECODIFICA os dados, mas NÃO VERIFICA A ASSINATURA. Isso significa que você não pode confiar na autenticidade dos dados. Use isso apenas para inspecionar ou extrair informações de fontes confiáveis. A verificação de assinatura requer uma chave secreta e bibliotecas criptográficas.

A biblioteca PyJWT é o padrão para lidar com JWT em Python. Instale com pip install pyjwt.

Use a biblioteca jsonwebtoken para uma abordagem robusta em Node.js. Instale com npm install jsonwebtoken.
